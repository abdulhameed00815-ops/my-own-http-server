A post request inheretely comes with a request body. Most prestigious web framework validate the incoming request body using a predefined model (class) that is passed as a parameter in the endpoint function. The initial approach we have to take is as follows:
1-check if the function signature contains a param of type "class".
2-if so, validate only if the request body is of type "application/json".
3-then we validate the json request body to see if it matches with the model.
4-we turn the json body into a class instance, inject it to the func and voila.
